# BIN-ARG2.0
Hoy les traigo esta herramienta multifuncional , que se trata de un menú de 3 opciones , quieres aprender carding opción 1 ,genera checkers de Argentina opción 2, genera checkers de diferentes paises opción 3.
Ya actualizado a la versión 1.8 Beta con dos opciones más en la 4 opción descarga el pdf Carding.pdf si deseas saber que es carding y en la opción 5 le agregue un bot de telegram que da bins funcionales de cualquier país.
